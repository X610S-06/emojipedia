# Emojipedia
Emojis description using ReactJS.

# Clone and Use

  Open Terminal 
  
    git clone 'https - copied url'

  Change directory  

    cd 'directory name'

  Open all files on VS code  

    code .

  Install dependencies

    npm install

  Run the website on localhost

    npm start
